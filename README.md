# UniSync - Campus Life Platform

**UniSync** is an all-in-one campus life platform designed to synchronize essential student services into a single, seamless web application. It bridges the gap between daily campus needs—from tracking lost items to finding study partners—creating a more connected and efficient university experience.

🔗 **Live Demo:** [https://unisyncc.netlify.app/](https://unisyncc.netlify.app/)

## 🚀 Overview

UniSync integrates four core modules to address common challenges faced by students living and studying on campus. It serves as a centralized hub for logistics, social study connections, and personal productivity.

## ✨ Key Features

### 1. 🔍 Lost & Found
A campus-wide tracking system to report and recover lost belongings.
* **Report Lost Items:** Submit details about missing items.
* **Search & Match:** Check the database for found items.
* **Real-time Updates:** Status tracking for reported items.

### 2. 🍔 Campus Food Tracker
A real-time monitor for cafeteria availability to help students save time.
* **Live Menu Updates:** See what is currently being served.
* **Availability Alerts:** Get notified when popular items are running low.
* **Student Contributions:** Community-driven updates on food status.

### 3. 📚 Study Group Finder
A social feature to connect students with similar academic goals.
* **Course-Based Matching:** Find partners studying the same subjects.
* **Session Scheduling:** Organize study times based on mutual availability.
* **Resource Sharing:** Platform for sharing notes and materials.

### 4. 🧠 BrainBrew
A personalized productivity tool designed to optimize study sessions.
* **Energy Level Detection:** Customizes sessions based on how you feel (Energized, Normal, or Tired).
* **Smart Focus Timer:** Includes Pomodoro, Deep Focus, and Extended Session modes.
* **Ambient Environment:** Curated playlists and background settings to enhance concentration.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Hosting:** Netlify
* **Icons:** FontAwesome (or similar library used)



## 📦 Installation & Setup

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/your-username/unisync.git](https://github.com/your-username/unisync.git)
    ```
2.  **Navigate to the project directory**
    ```bash
    cd unisync
    ```
3.  **Open the project**
    * Simply open `index.html` in your browser.
    * *Or, if using a local server:*
        ```bash
        # Example using python
        python -m http.server 8000
        ```
